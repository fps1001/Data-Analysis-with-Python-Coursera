Practice Project: Insurance Cost Analysis

Estimated time needed: 75 minutes

In this project, you have to perform analytics operations on an insurance database that uses the below mentioned parameters.

Parameter Description Content type
age Age in years integer
gender Male or Female integer (1 or 2)
bmi Body mass index float
no_of_children Number of children integer
smoker Whether smoker or not integer (0 or 1)
region Which US region - NW, NE, SW, SE integer (1,2,3 or 4 respectively)
charges Annual Insurance charges in USD float

Objectives

In this project, you will:

  • Load the data as a pandas dataframe
  • Clean the data, taking care of the blank entries
  • Run exploratory data analysis (EDA) and identify the attributes that most affect the charges
  • Develop single variable and multi variable Linear Regression models for predicting the charges
  • Use Ridge regression to refine the performance of Linear regression models.

Handle missing data:

  • For continuous attributes (e.g., age), replace missing values with the mean.
  • For categorical attributes (e.g., smoker), replace missing values with the most frequent value.
  • Update the data types of the respective columns.
  • Verify the update using df.info().
In[8]:
# Age Dtype to int and missing values to mean.
# Ensure age is numeric first (coerce invalid entries to NaN), then compute mean and fill NaNs.
data['age'] = pd.to_numeric(data['age'], errors='coerce') # coerce means invalid parsing will be set as NaN
mean_age = data['age'].mean()
# Filling NaN values with mean age
data['age'] = data['age'].fillna(mean_age)
# Converting age to int
data['age'] = data['age'].round().astype(int)
In[9]:
# Smoker to int and missing values to most frequent value.
most_frequent_smoker = data['smoker'].value_counts().idxmax()
# Avoid pandas FutureWarning from inplace on a Series slice by assigning the filled Series back to the DataFrame
data['smoker'] = data['smoker'].fillna(most_frequent_smoker)
# Now convert smoker column to integer (safe because NaNs were filled)
data['smoker'] = data['smoker'].astype(int)
